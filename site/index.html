Metered AI coding is the new cloud bill: invisible while you work, shocking at month-end. aireceipts turns the transcripts already sitting on your disk into itemized receipts — so you learn which habits burn money (re-running a failing command five times: $0.08 and 3m45s), which model your trivial turns actually need, and what each PR cost to produce. Then it helps you fix it: every receipt ends with a paste-ready handoff that tells your agent what to do cheaper next time.

Why you can believe it

  1. No fabricated dollars, ever.A model without a cited, dated price row renders tokens-only — never a guess.
  2. Every price cited with URL + date + excerpt.CI checks the citations and that each one is still live.
  3. Comparisons are arithmetic, not predictions.“Same tokens on X” re-prices the identical token counts — it never claims another model would have done the job.
  4. Deterministic: same transcript in, byte-identical receipt out.Golden-tested on every commit, 10× under a frozen environment.

Diagnostics-only telemetry, fully disclosed, one env var to kill — and your transcripts never leave your machine.
