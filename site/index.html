// 03 · when the PR ships

The receipt sticks with the PR.

aireceipts pr --post attaches the observable floor of the sessions behind a branch to its pull request — one comment, kept current across pushes. A drop-in CI check can require every PR to carry one. The observed usage behind a change becomes as visible as what it changed.
